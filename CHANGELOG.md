# Changelog

## v2.2.0

### New Stacks
- **Mental Models** (`/cube-models`): OODA Loop (Boyd), Circle of Competence (Munger/Buffett), Jobs to Be Done (Christensen), Root Cause/5 Whys (Ohno), Leverage & 80/20 (Pareto), Opportunity Cost (Bastiat)
- **Bias** (`/cube-bias`): Incentive Bias (Munger), Availability (Kahneman), Social Proof (Cialdini), Anchoring (Tversky), Loss Aversion, Confirmation Bias
- **Executive** (`/cube-executive`): CEO, CFO, COO, CMO, CPO, CTO — same problem through six C-suite lenses
- **Thinking Hats** (`/cube-hats`): de Bono's Six Thinking Hats — White, Red, Green, Blue, Black, Yellow
- **Negotiation** (`/cube-negotiate`): Tactical Empathy (Voss), Principled Negotiation (Fisher/Ury), Influence (Cialdini), Conflict Modes (Thomas-Kilmann), Prospect Theory (Kahneman), Bargaining Power (Shell)
- **Systems** (`/cube-systems`): Feedback Loops (Meadows), Leverage Points (Meadows), System Archetypes (Senge), Causal Loop Mapping (Sterman), Emergence (Kauffman/Gladwell), Constraints (Goldratt)

### New Features
- **Plugin package**: Downloadable `.zip` for Claude Code / Cowork plugin install
- **`build-plugin.sh`**: Script to assemble plugin zip from source

- **Adversarial** (`/cube-adversarial`): Red Team (Zenko), ACH (Heuer/CIA), Devil's Advocacy, Chaos Engineering (Netflix), Attack Trees (Schneier), Antifragility (Taleb)

### New Features
- **`/cube-enhance`**: Meta-skill that analyzes any other skill through 16 Cube positions
